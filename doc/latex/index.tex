{\ttfamily $\ast$} — On some Linux distributions such as Ubuntu, the directory in which the v0idengine library is located ({\ttfamily /usr/local/lib/}) is not a default path. Execute the command below and run the game again\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{export LD\_LIBRARY\_PATH="{}\$LD\_LIBRARY\_PATH:/usr/local/lib"{}}

\end{DoxyCode}
